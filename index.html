About me

I am a self-taught software-developer currently living in Nottingham. Most of my skills are in web-development though I am very willing to explore other areas of programming as the universal logic of systems is more fulfilling to me than the particularity of any language or framework. I have knowledge of both front and back end design patterns and am capable of producing full-stack applications. I am looking forward to finding my first professional experience and am willing to travel to acheive this. Please peruse my projects below.

Alchem.io

My most recent project. A user-submitted cocktail-recipe site. It uses API calls to communicate with an AWS Lambda function that places into and draws data from an SQL database. This is my most up-to-date example of code quality.

Mandal.io

My first project. A tool for drawing mandalas on an HTML canvas. It will symmetrically mirror your inputs across the axes, you can change colours and brush size. The bucket fill is the most complex code in this and I spent a long time researching the algorithm for it. My code quality has improved a lot since making this but people say its my most fun project.

Knight's tour project

This project uses an algorithm I found by searching through mathematical papers to solve the knights tour for arbritrary board sizes. The code that fits the variable board size to the screen-size is surprisingly almost as interesting as the algorithm itself.

Front-end aesthetics project
